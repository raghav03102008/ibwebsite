# IB Study Hub

A modern, responsive dashboard interface designed for IB (International Baccalaureate) students to organize their study workflow in a single place.

> This repository currently contains the initial dashboard interface and layout structure. Functional learning tools and data management features are planned for future development.

## Features

### Dashboard Home

* Clean dark-themed interface
* Responsive layout
* Subject overview cards
* Quick navigation system
* Interactive hover effects

### User Interface

* Modern design system
* Consistent color palette
* Smooth animations and transitions
* Card-based layout
* Mobile-friendly experience

### Subjects Included

* Mathematics
* Physics
* Chemistry
* Biology
* Economics
* History
* English A
* Environmental Systems & Societies (ESS)
* Theory of Knowledge (TOK)
* Extended Essay (EE)
* Internal Assessments & CAS

## Built With

* HTML5
* CSS3
* Vanilla JavaScript

No external frameworks or dependencies are required.

## Project Structure

```text
project/
│
└── index.html
```

The current version is intentionally lightweight and runs entirely in the browser.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/ib-study-hub.git
```

Navigate to the project folder:

```bash
cd ib-study-hub
```

Open `index.html` in your browser.

No installation or build process is required.
